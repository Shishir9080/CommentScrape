# CommentScrapper

Automated tool designed to pick out interesting HTML comments from a website.  

__Example Run__:

```
$ python3 ./scrape_page.py -c -t "https://543hn.com"
```

![](markdown/example_run.png)

Requirements:
- requests (requests)
- beautiful soup (bs4)
- validators (validators)
- colorama (colorama)
