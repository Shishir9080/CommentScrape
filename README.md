# CommentScraper

Automated tool designed to pick out interesting HTML comments from a website.  

__Example Run__:

```
$ python3 ./scrape.py -Cc "https://543hn.com"
```

![](![Example Run](screenshots/example_run.png)

Requirements:

- requests (`requests`)
- beautiful soup (`bs4`)
- validators (`validators`)
- colorama (`colorama`)
